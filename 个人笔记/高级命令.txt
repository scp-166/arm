find [路径] 选项 查找条件

find . -name "*.txt"
路径默认是 . 

find . -mtime -1
查找一天内有变动的文件


grep
目的: 查找文件中符合条件的字符串
grep [选项] [查找模式] [文件名/路径]


查找字符串时希望显示如下内容
1. 所在的文件名（默认grep查找时已经显示目标文件名）
2. 所在的行号（使用-n选项）

grep -rn '字符串' 文件名
: r(recursive) 递归查找
: n(number)  显示目标位置的行号
字符串: 要查找的字符串
文件名: 要查找的目标文件，如果是*表示查找当前目录下的所有文件和目录

：w 全字匹配(单独的字符串才会查找出来)

file
目的: 识别文件类型
格式 file 文件名

file *  查看当前所有文件的类型
file demo/*  查看demo目录下所有文件的类型

which
这条命令主要是用来查找系统PATH目录下的可执行文件。
如果在PATH没有添加需要的命令，则无法查找

whereis
这个命令可以用来查找二进制（命令）、源文件、man文件。
与which不同的是这条命令可以是通过文件索引数据库而非PATH来查找的，
所以查找的面比which要广。

解压缩
单个文件的解压缩用 gzip 和 bzip
多个文件和目录使用 tar

gzip
- 选项
    - l  列出压缩文件的内容
    - k(keep)  在压缩或解压时，保留输入文件
    - d(decompress)  将压缩文件进行解压缩
- 查看 
    gzip -l 压缩文件名
- 解压
    gzip -kd 压缩文件名
    如果没有k，那么压缩文件会被删除
- 压缩
    gzip -k 文件 压缩后的文件名
    如果没有k，那么原文件会被删除
- 注意
    - 相同的文件内容，如果文件名不相同，压缩后的大小也不同
    - gzip只能压缩单个文件，不能压缩目录
    
bzip2
- 类似gzip， 没有-l参数

gzip适合单个小文件压缩
bzip2适合单个大文件压缩


tar
- 选项
    - c 表示打包
    - z 用gzip方式
    - j 用bzip2方式
    - v(verbose) 显示详细记录
    - f 后面接文件名
    - t 查看
    - C 解压指定路径名
- 压缩
    tar -czvf 压缩文件名 目录名
    tar -cjvf 压缩文件名 目录名
    比如 tar czvf dira ./dira
    横杠可以不用写
- 查看
    tar -tvf 压缩文件名
- 压缩
    tar -xzvf 压缩文件名
    tar -xjvf 压缩文件名
        默认是当前目录
    tar -xzvf 压缩文件名 -C 指定目录
    tar -xjvf 压缩文件名 -C 指定目录
